### Analysis Notes
The concept_context tool identified only one concept (HOLY/H6942) in Exodus 20:8, since the verse's key action verb is qadash ("to keep it holy"). The tool does not identify zakar ("remember") or shabbath as separate indexed concepts, so the HOLY concept provides the primary cross-reference network.

The connection between Exo 20:8 and Exo 20:11 in the same chapter is significant: v.8 commands keeping the Sabbath holy; v.11 explains that God Himself "hallowed" (qadash) the day -- the same verb in both verses. God commands what He Himself has already done.

The pre-Sinai connection to Exo 16:23 ("the rest of the holy sabbath") confirms the holiness of the Sabbath was already established before the Decalogue was formally given at Sinai.
